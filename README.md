# Latex Template

This is the template that I use to create homework documents for college. Feel free to modify or clone it for use in your own documents.

**Default Settings**

```
title: "Practical Assignment on Integrals"
name: "Jon Doe"
subject: "Advanced Mathematics"
semester: "2025-II"
professor: "Dr. Smith"
university: "Example University"
career: "Engineering"
department: "Department of Mathematics"
where: "Room 101"
toc: true
icons-dir: images/
```

Web viewer server
_https://pypi.org/project/reloadserver/_
```
python3 -m reloadserver --watch viewer.html --watch build/report.pdf
```
